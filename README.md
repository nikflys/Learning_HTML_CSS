# Learning HTML & CSS

Repositorio de práctica mientras aprendo desarrollo web desde cero. Cada carpeta es un tema puntual con su propio HTML y CSS.

## Estructura

### `html_learning/`
Conceptos de HTML, de lo más básico a lo más específico.

| Carpeta / Archivo | Qué se ve |
|---|---|
| `parrafos_y_encabezados.html` | Etiquetas h1–h6 y párrafos. El h1 importa para SEO. |
| `listas.html` | Listas sin orden (`ul`) y sus ítems (`li`). |
| `img.html` | Imágenes y el atributo `alt`. |
| `rutas.html` | Diferencia entre rutas absolutas y relativas. |
| `forms.html` | Tipos de input: text, email, password, checkbox, radio, submit, file, date, color. |
| `tablas/` | Estructura de tablas: `thead`, `tbody`, `tfoot`, `th`, `td`, `colspan`. |
| `enlaces basic y avanzados/` | Links simples y formulario de contacto completo con `fieldset`, `select`, `datalist`, `textarea`. |
| `html semantico/` | Uso correcto de `header`, `nav`, `main`, `section`, `aside`, `footer`. |
| `accesibilidad web/` | Atributos `role` y `aria-*` para lectores de pantalla. |
| `comentarios metatags y favicons/` | Metatags para SEO, Open Graph para redes sociales, y favicons. |
| `details y summary/` | Acordeón nativo de HTML sin necesidad de JS. |
| `lazy log/` | Carga diferida de imágenes con `loading="lazy"`. |
| `text area/` | `textarea`, `fieldset`, `legend`, `select` y `datalist`. |
| `videos/` | Etiqueta `video` con sus atributos y subtítulos con `track`. |

---

### `css_learning/`
Propiedades de CSS organizadas por tema.

| Carpeta | Qué se practica |
|---|---|
| `selectores/` | Selectores de elemento, clase (`.`) e id (`#`). |
| `bordes/` | `border`, `border-radius`, `border-image`. Truco de solo `border-bottom`. |
| `box sizing/` | Diferencia entre `content-box` y `border-box`. |
| `colorizacion/` | Formas de definir colores: nombre, `rgb()`, `rgba()`, HEX. Shorthand de `padding`. |
| `fondos/` | `background-image`, `background-size`, `background-position`, `background-attachment`, `linear-gradient`, `radial-gradient`. Truco para oscurecer imágenes. |
| `padding y margin/` | Shorthand de padding y margin. `margin: auto` para centrar. |
| `propiedades_de_texto/` | `font-family`, `font-size`, `font-weight`, `font-style`, `line-height`, `letter-spacing`, `text-align`, `text-decoration`, `text-transform`. Fuente local (Montserrat). |
| `sombras/` | `box-shadow`, `text-shadow`, `filter: drop-shadow()`. Diferencia entre box-shadow y drop-shadow. |
| `unidades de medida/` | `px`, `%`, `em`, `rem`, `vh`, `vw`, `vmin`, `vmax`, `ex`. |
| `width_y_height/` | `width`, `height` y sus variantes `min-`/`max-`. |
| `ejercicios/` | Cards de perfil combinando todo lo anterior. |
